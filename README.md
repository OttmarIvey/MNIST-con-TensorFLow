# MNIST con TensorFLow
En este proyecto se abordara el problema MNIST, el cual consiste en un conjunto de 70000 imagenes en escala de grises, que contienen un numero escrito a mano. El objetivo será crear un modelo de aprednizaje automático que sea capaz de distinguir el numero en cada imagen. Esto utilizando la paquetería de TensorFLow en Python.
